# 날라앱스 커머스 v1.2.0-rc.2 릴리즈 기록

- 배포일: 2026-07-15
- 상태: Release Candidate
- 핵심 변경: 날라앱스 메시지 무료화와 iwinv 재판매 중앙 서비스

## 날라앱스 메시지 정책

- 커머스용·독립형 애드온 모두 무료
- 유료 애드온 시리얼 없음
- 문자 발송 사업자는 iwinv 단일 고정
- 고객이 자신의 iwinv 계정이나 인증정보를 입력하는 기능 없음
- 고객은 app.nal.la에서 메시지 잔액을 충전
- 고객별 판매단가·잔액·충전·차감·환불·발송내역은 app.nal.la에서 관리
- iwinv 조직 인증정보와 발신번호·알림톡 채널은 app.nal.la에서 중앙관리

## 아키텍처

```text
고객 WordPress 무료 애드온
  → app.nal.la 메시지 API·고객 지갑
  → 고객별 iwinv 발송조직
  → iwinv 메시지 서비스
```

## 보강사항

- HMAC 요청서명·요청시각·고유 요청 ID 검증
- 고객 사이트 재등록시 기존 잔액과 원장 보존
- 잔액 차감과 원장 기록 DB 트랜잭션 처리
- iwinv 요청 실패시 자동환불과 월 사용액 복구
- app.nal.la 최근 발송내역과 충전·차감 원장 화면
- 무료 애드온을 EDD 상품 ID 0과 무료 라이선스 정책으로 카탈로그 투영
- app.nal.la 결제완료 후 메시지 지갑 적립용 서버 이벤트 계약

## 검증

- 전체 소스 회귀시험 PASS
- PHP 129개·JavaScript 5개 소스 구문검사 PASS
- 공개 ZIP PHP 214개·JavaScript 9개 검사 PASS
- 내부 서버 ZIP 포함 PHP 228개·JavaScript 9개 검사 PASS
- 개발판·EDD 운영판 공개 산출물 16개 재현 빌드 일치
- 내부 산출물 19개 재현 빌드 일치
- 비밀정보 정적 탐지 0건

## 아직 필요한 외부시험

- iwinv 콘솔 API 가이드의 실제 요청·응답 필드 매핑
- 실제 고객별 발송조직과 발신번호 등록
- 실제 SMS·LMS·MMS 발송과 결과코드 확인
- 실제 알림톡 비즈니스 채널·템플릿 승인
- app.nal.la 결제와 고객 지갑 자동적립
- 실제 WordPress 스테이징 설치·업데이트

정식 Stable 승격 전에는 운영 사이트에 직접 설치하지 않습니다.
