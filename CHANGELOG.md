## [1.2.0-rc.1] - 2026-07-15

### 상태

- 설치 테스트용 Release Candidate
- 정식 Stable 아님

### 배포 구성

- 날라앱스 커머스 Core
- 커머스용 공식 애드온 5개
- 동일 저장소에서 생성한 독립 플러그인 5개
- 개발 오픈 라이선스 빌드
- EDD 강제 운영 빌드

### 주요 변경

- 각 애드온 저장소를 단일 원본으로 사용하면서 커머스용과 독립형 두 배포본 생성
- 제품별 부트스트랩·라이선스·업데이트 상태 격리
- 두 배포형 동시 활성화 시 중복 Hook·클래스·DB 실행 전 차단
- 독립형 EDD 라이선스·도메인·만료·7일 통신장애 유예
- 독립형 GitHub 카탈로그와 ZIP SHA-256 검증
- Core는 플랫폼 기능만 유지하고 애드온 실행코드는 포함하지 않음

### 검증

- 전체 자동테스트 PASS
- PHP 119개·JavaScript 5개 소스 구문검사 PASS
- 배포 ZIP 전체 PHP 212개·JavaScript 9개 구문검사 PASS
- 개발판·EDD 운영판 각각 16개 산출물 재현 빌드 일치
- 비밀정보 정적 탐지 0건
- 공개 패키지 전체 소스 ZIP 포함 0건

### 제한사항

- 실제 WordPress 스테이징 설치 전
- 실제 app.nal.la EDD 라이선스 상품 연결 전
- GitHub Release 원격 업데이트 실연동 전
- SMS·알림톡·SMTP 실계정 발송 전
